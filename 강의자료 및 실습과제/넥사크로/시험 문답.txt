this.divCommand_btnSearch_onclick = function(obj:nexacro.Button,e:nexacro.ClickEventInfo)

{ 코드 구현 };

this : 현재 작업중인 form 을 의미함

divCommand_btnSearch : 클릭하는 버튼의 아이디

onclick : 이벤트 종류

function : 클릭시 작동될 콜맥함수임

obj : 클릭한 버튼 객체를 의미함

e : 이벤트 정보를 가진 객체임

서술형



Button 컴포넌트의 사용법과 속성 지정 방법을 기술하시오. (10점)



Component 툴바에서 Button 툴버튼 클릭

폼 디자인 영역에 드래그 또는 클릭함

화면 오른쪽의 Properties 뷰에서 필요한 속성(id, text, font, color 등)에 값 지정함




컴포넌트(Component)와 데이터셋(Dataset)을 연동하는 두가지 방법에 대해 기술하시오 (25점)


첫번째 폼 디자인에서 하단의 'Invisible Object' 에 있는 데이터셋 아이디를 컴포넌트로 드래그 드롭함

두번째, 컴포넌트를 선택한 다음 Properties 에서 'innerDataset' 속성에 연결할 데이터셋을 지정함





넥사크로 데이터셋(Dataset) 생성 방법에 대해 설명하시오. (25점)


Component 툴바에서 Dataset 툴버튼 클릭

폼 디자인 영역에 아무곳이나 클릭함

Invisible Object 영역에 추가된 데이터셋 객체가 표시됨





컴포넌트에 이벤트를 연결하여 이벤트함수를 생성하는 방법에 대해 서술하시오.(40점)


- 클릭(Click) 이벤트의 경우는 폼 디자인 화면에서 컴포넌트를 더블클릭하면 이벤트함수가 생성됨

- 폼 디자인화면에서 컴포넌트를 선택한 다음, 화면 오른쪽의 Properties 에서 'Event' 탭을 선택

- 이벤트 속성들 중에서 사용할 이벤트속성을 선택

- 선택한 이벤트 속성의 자동 생성될 이벤트 함수명을 선택하면 '스크립트'창에 소스가 생성됨
